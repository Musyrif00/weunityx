# WeUnityX

React Native social media app with live streaming, video calls, and Web3 wallet integration.

## Features

- Social posts, stories, comments, likes
- Live streaming (Agora)
- Video/voice calls
- Real-time chat
- Multi-chain wallet

## Tech Stack

React Native + Expo 49 | Firebase | Agora RTC | Moralis API

## Quick Start

```bash
npm install
npx expo start --dev-client
```

## Setup

Requires Firebase, Agora, and Moralis configuration.
