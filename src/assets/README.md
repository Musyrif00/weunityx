# WeUnityX Assets

This directory contains all the assets for the WeUnityX app:

- **icons/**: App icons and UI icons
- **images/**: Images, backgrounds, and graphics
- **splash/**: Splash screen assets
- **adaptive-icon/**: Android adaptive icon assets

## Asset Requirements

### Icons

- App icon: 1024x1024 PNG
- Favicon: 32x32 PNG
- Notification icon: 96x96 PNG (Android)

### Splash Screen

- Splash image: 1284x2778 PNG (iOS)
- Adaptive splash: 1080x1920 PNG (Android)

### Placeholder Assets

For development, you can use placeholder images from:

- https://ui-avatars.com/ (for user avatars)
- https://picsum.photos/ (for random images)
- https://i.pravatar.cc/ (for user profile pictures)
